How Web Server Works
A web server works using for the commutioncation between client and server using the Hypertext Transfer Protocol (HTTP) by storing, processing and delivering web pages to clients.

Mostly, pages delivers HTML documents which includes images, style sheets and scripts. Furthermore, server also send some useful information likes last modification time and its MIME type. The MIME type tell browsers what it should do to display the received documents. For example, MIME type is text/html will tell browser to render it as text.
